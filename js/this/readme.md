# this的绑定规则
1. 默认绑定 --- 函数在哪个词法作用域里生效,this就指向哪里

2. 隐式绑定 --- 当函数被一个对象所拥有，再调用时，此时this会指向该对象

3. 隐式丢失 --- 当函数被多个对象链式调用时，this指向引用函数的对象

4. 显示绑定 --- call,apply,bind

5. new绑定   

# 箭头函数 (ES6新增函数)
箭头函数没有this这个概念，写在箭头函数中的this也是它外层函数的this
